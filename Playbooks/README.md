## Scheme comparison (at a glance) 🧠

These are **starting points**.

- “CFB teams to check” = real-world programs commonly associated with the scheme family (verify in-game).
- “NFL analogs” = teams that commonly use similar principles (varies by coordinator/season).

| Scheme                                     | Typical personnel | Core idea                                 | CFB teams to check first (verify in-game)                                                             | NFL analogs (approx)                                                                               |
| ------------------------------------------ | ----------------- | ----------------------------------------- | ----------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| [Air Raid](air-raid/README.md)             | 10 / 11           | Spacing + quick game + repeatable reads   | Texas Tech, Washington State, Mississippi State, Oklahoma (spread pass influence)                     | Chiefs (spread/quick), Bengals (shot + spread), Bills (spread), Cardinals (Air Raid-era influence) |
| [Pro Style](pro-style/README.md)           | 12 / 21           | Run game sets up PA, win in red zone      | Michigan, Georgia, Alabama, Notre Dame, Wisconsin, Iowa                                               | 49ers, Rams, Lions, Packers (West Coast-derived), Browns                                           |
| [Spread](spread/README.md)                 | 10 / 11           | Box counts + RPOs + tempo                 | Oregon, Ohio State, Ole Miss, TCU, Baylor, Tennessee                                                  | Eagles, Bills, Chiefs, Cowboys, Texans                                                             |
| [Veer and Shoot](veer-and-shoot/README.md) | 10 / 11           | Wide splits + access throws + inside zone | Tennessee (Heupel), UCF (Heupel-era), Baylor (Briles-tree concepts), Houston (tempo spread influence) | Dolphins (wide splits/quick), Rams (spacing), Bengals (spread), Chiefs (spacing)                   |
| [West Coast](west-coast/README.md)         | 11 / 12 / 21      | Rhythm throws + YAC + layered concepts    | Stanford (historical), Utah, Notre Dame (pro-style/WCO overlap), BYU (timing concepts)                | 49ers, Packers, Rams, Vikings, Dolphins                                                            |
| [Option](option/README.md)                 | 11 / 20 / 21      | Read/pitch creates +1 run advantage       | Army, Navy, Air Force, Georgia Tech (historical), New Mexico (option elements)                        | Ravens (QB run), Eagles (option packages), Bills (QB run), Bears (QB run packages)                 |
| [Run and Shoot](run-and-shoot/README.md)   | 10 / 11           | Option routes + leverage reads            | Hawaii (June Jones-era), SMU (historical), Houston (run-and-shoot influence)                          | Patriots (choice routes), Chiefs (option routes), Packers (option routes)                          |

## Notes

- “Personnel” uses common football notation:
  - 10 = 1 RB, 0 TE, 4 WR
  - 11 = 1 RB, 1 TE, 3 WR
  - 12 = 1 RB, 2 TE, 2 WR
  - 21 = 2 RB, 1 TE, 2 WR
  - 22 = 2 RB, 2 TE, 1 WR
- Each guide includes a section for “Teams in CFB26 using this playbook.” That section is intentionally editable because assignments can change by patch/roster.

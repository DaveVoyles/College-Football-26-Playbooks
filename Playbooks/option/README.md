## Quick definition 🧠

“Option” isn’t one formation—it’s a **decision-based run game**.

Instead of blocking every defender, you **leave one unblocked** and make him wrong by:

- 🏈 Giving the ball (Dive)
- 🏃 Keeping the ball (Keeper)
- ⚾ Pitching (Pitch/Option)

```text
       WR            OL            WR
              SB     QB     SB
           (Slot)          (Slot)
                     FB
```

In games, option playbooks typically blend:

- **spread option** (gun reads, QB keepers, RPO tags)
- **traditional option** (speed option, triple option, flexbone variants)

## The identity (one sentence) 🎯

**Create a numbers advantage in the run game every snap, then hit constraint passes when the defense overreacts.**

---

## Pros ✅ and cons ⚠️

```diff
+ PRO: Portable Run Game (reads replace blocks).
+ PRO: Forces Discipline (punishes aggressive defense).
+ PRO: Controls Pace (shorten games, limit possessions).
+ PRO: +1 Numbers Advantage (QB creates the extra man).
+ PRO: Beats Light Boxes (two-high shells get gashed).

- CON: QB Durability (taking hits is part of the job).
- CON: Hard to Chase Points (clock + predictability).
- CON: Vulnerable to "Scrape Exchange" (requires passing answers).
- CON: Turnover Risk (pitch timing must be perfect).
```

## Typical personnel 🧩

Option systems vary more than other styles. Common groupings:

- **11 personnel**: 1 RB, 1 TE, 3 WR
  - Spread option base (zone read, RPO, QB keep).
- **20 personnel**: 2 RB, 0 TE, 3 WR
  - Great for speed option and misdirection.
- **21 personnel**: 2 RB, 1 TE, 2 WR
  - More downhill QB run + lead concepts.

### Personnel cheat table

| **Personnel** | **On-field**       | **Why you’d use it in Option**                          |
| ------------- | ------------------ | ------------------------------------------------------- |
| 11            | 1 RB / 1 TE / 3 WR | Spread option base: zone read, RPO, QB keep             |
| 20            | 2 RB / 3 WR        | Speed option/misdirection; pitch relationships in space |
| 21            | 2 RB / 1 TE / 2 WR | More downhill; lead blockers; short yardage QB run menu |

Traditional option variants sometimes use heavier looks (not always labeled cleanly in-game), but the “idea” is:

- multiple backfield threats
- a tight surface for blocking angles

### Role archetypes

- **QB:** speed/acceleration, ball carrier moves, durability; plus “good enough” short accuracy.
- **RB1 (dive back):** tough, inside runner.
- **RB2 (pitch back / slot back):** speed in space.
- **WRs:** blockers first, then vertical threats on play-action.
- **TE/H-back (if used):** edge setter.

## QB operation (timing + progressions) ⏱️

### Option fundamentals (the rules that make it work) 🧠

### Rule #1: know who you’re reading

Every option has a read key.

Common read keys:

- backside defensive end (zone read)
- play-side end/outside linebacker (speed option)
- first man outside the tackle (triple option)

### Read-key cheat table

| **Play family** | **Typical read key**    | **Your decision**                              |
| --------------- | ----------------------- | ---------------------------------------------- |
| Zone read       | Backside DE             | Give vs squeeze/crash • Keep vs slow/play wide |
| Speed option    | Play-side edge (DE/OLB) | Pitch if he takes QB • Keep if he takes pitch  |
| Triple option   | Dive key + pitch key    | Give/keep/pitch based on first commitment      |

If you can’t identify the key pre-snap, don’t call the play.

### Rule #2: don’t “hope” — decide

Option runs fail when you hesitate.

```diff
+ GREEN: Key sits or widens -> GIVE the Dive
- RED:   Key crashes down   -> PULL and Keep
```

Commit to give/keep/pitch based on the key’s first step.

### Rule #3: use constraint plays

Option is a conversation with the defense.

If they overplay the QB, give it.
If they overplay the dive, keep it.
If they overplay the pitch, turn it into a play-action shot.

## Core run game families 🏃

### 1) Inside zone read

**What it is:** inside zone with a backside read.

**When it’s great:**

- vs aggressive ends
- when defenses chase your RB hard

**Coaching point:** don’t drift backwards on the keep—get vertical.

### 2) Speed option

**What it is:** QB attacks edge with a pitch relationship.

**When to call:**

- vs wide fronts
- after inside reads make defenders squeeze

```diff
+ GREEN: Edge Defender crashes QB -> PITCH immediately
- RED:   Edge Defender widens     -> KEEP and turn upfield
```

**Pitch habit:** pitch early enough to avoid hit-sticks, late enough to force the defender to commit.

### 3) Triple option (if in your book)

Classic triple option typically creates:

- dive
- QB keep
- pitch

It’s deadly if you can keep your spacing and make clean decisions.

```diff
! 1ST READ (Dive Key)
+ key sits -> GIVE dive

! 2ND READ (Pitch Key)
+ key takes QB    -> PITCH
- key takes Pitch -> KEEP
```

### 4) QB power / QB counter

These aren’t “option” by definition, but they complement option identities perfectly.

Use them when defenses start playing assignment football and you need a straight-up numbers run.

### 5) Midline / inside veer variants

If your playbook includes these, they punish certain fronts by reading interior defenders.

## Core pass game families / concept menu ⚙️

Option offenses need a **small, brutal passing menu**:

### 1) Play-action shot (post / seam)

Call it after 2–3 successful option runs.

Targets:

- seam behind linebackers
- post behind single-high

### 2) Quick game

- slants
- hitches
- quick outs

Use quick game when corners give free access.

### 3) RPO tags (for spread option variants)

- bubble/now
- glance

These punish overhang defenders who are trying to fit the run.

## Defensive looks and your counters 🛡️

### They “scrape exchange” the read

What it means: a defender replaces the read player to take away the keep.

Your answers:

- call a different read (change the key)
- use QB power/counter (no read)
- hit quick pass/RPO behind the exchange

### They set hard edges

- run inside (dive, inside zone)
- call misdirection (counter)
- use play-action to punish safeties triggering

### They spy the QB

- throw to the vacated area
- run your RB at the lighter box

## Formation packages (how to pick what “type” of option you are) 📐

Most option playbooks can be played in two broad ways. Pick one as your base so your calls feel coherent.

### Package A: Spread option (gun-based)

**What it looks like:** 11/10 personnel, lots of gun sets.

**What it majors in:**

- inside zone read
- RPO bubble/glance
- QB keepers (speed option, QB draw)

**Why it’s popular in-game:**

- easier passing on obvious downs
- easier to mix tempo
- simpler to identify the read key

### Package B: Traditional option (tight surface)

**What it looks like:** condensed formations, more backs, stronger angles.

**What it majors in:**

- triple option
- speed option
- veer/midline variants

**Why it’s scary:**

- angles are cleaner
- the pitch phase can stress defenses that “overplay” the QB

If you’re not sure which one your roster supports, start with spread option. It’s the easiest to keep functional when you’re down on the scoreboard.

## Pitch mechanics (how to avoid the two worst outcomes) ⚾

The two worst outcomes on option plays are:

1. **A huge loss** because you pitched too late into a hit.
2. **A turnover** because the pitch was rushed or thrown into traffic.

### A simple pitch rule

- If the force defender commits to the QB: pitch.
- If the force defender stays wide and slow-plays: keep and get vertical.

### Spacing rule (pitch relationship)

Try to maintain a spacing where your pitch back is:

- not so close that one defender can tackle both of you
- not so far that the pitch becomes a slow, floating throw

If you feel like pitches are always late, it usually means you’re letting the QB drift sideways too long. Turn upfield sooner.

## Constraint play menu (your “answers” when they sell out) 🧰

Option is only unstoppable when you punish overreactions.

Here’s a practical constraint menu to keep in your favorites.

### If they crash the dive (to stop the handoff)

- keep the read (QB keep)
- call QB power/counter (no read)

### If they sit on the QB keep

- give the dive repeatedly
- call speed option toward the overplay

### If they fly to the pitch

- keep it and cut upfield
- hit play-action behind them (seam/post)

### If they widen hard to stop the perimeter

- run inside (dive, inside zone)
- call counter back inside

## Pre-snap plan (audibles + box count + motion) 🎮

Option offenses get wrecked when you call a read into a look that defeats it. Audibles prevent that.

Suggested audible set (example):

1. Inside zone read (base)
2. QB power (no read, downhill)
3. Speed option (perimeter)
4. RPO bubble / quick screen (if apex is tight)
5. Play-action shot (post/seam) (if safeties are nosy)

Pre-snap checklist:

- If the edge is already outside and waiting: don’t call speed option into it.
- If the box is light: don’t outsmart yourself—hand it off.
- If corners are off and safeties are deep: quick game is free yards.

## Weekly install plan (Dynasty-friendly) 📚

If you want the option game to feel automatic, practice in a simple progression.

### Week 1: inside zone read + one quick throw

- Learn to identify the backside end as the key.
- Pair it with one free-access throw (hitch or bubble).

### Week 2: add speed option

- Focus on pitch spacing and “turn upfield” timing.

### Week 3: add one shot play

- Use it only after you have forced the defense to trigger downhill.

### Week 4: add one counter / misdirection

- This is your answer to scrape exchange and hard edges.

## Troubleshooting (why option runs get blown up) 🧯

### Problem: “My QB keep gets tackled by someone unblocked.”

Likely cause:

- defense is scraping/exchanging or you’re reading the wrong key.

Fix:

- change which option play you’re calling (change the key)
- sprinkle QB power/counter

### Problem: “Speed option always loses yards.”

Likely cause:

- you’re running it into a wide, patient force defender.

Fix:

- call it after inside runs have squeezed the edge
- audible to inside zone if the edge is already set

### Problem: “I can’t pass out of this.”

Fix:

- simplify to one quick concept and one shot concept
- don’t call long progressions; use max protect or half-field reads

## Defensive-front cheat sheet (what options usually fit best) 🧾

This is a simplified way to choose calls without memorizing every defensive front name.

### If the edges are wide and patient

You’ll feel like speed option has no corner.

Better calls:

- inside zone/dive
- QB power (downhill)
- counter back inside

### If the edges are crashing hard

Better calls:

- zone read keep
- speed option (force them to play the pitch)
- play-action behind the crash

### If linebackers are scraping over the top

Better calls:

- change the read key (different option play)
- misdirection/counter
- quick RPO glance behind the scrape

## Situational notes (option can still win “passing situations”) 🧠

### 2-minute offense

Pure option can chew clock, so in 2-minute you typically shift toward:

- quick outs / hitches
- seams off play-action
- a screen to punish blitz

Keep one QB draw available as a surprise if the defense drops deep and vacates the middle.

### 4-minute offense (protecting a lead)

Option is excellent here:

- run inside reads to stay in bounds
- take safe constraint passes only when the defense overcommits

### Red zone

The QB run threat becomes stronger.

- QB power/keep
- quick rub pass
- play-action leak (if defense sells out)

## Post-Game Metrics (Did I run it right?) 📊

If you played a true Option game, your stats should look like this:

- **Rushing Attempts:** > 50 (Dominate the ground)
- **Passing Attempts:** < 15 (Only pass when necessary)
- **Time of Possession:** > 60% (Keep their offense off the field)
- **Yards Per Carry:** > 4.5 (Move the chains)

## Dynasty Recruiting Guide 🎓

### Quick targets

Option recruiting is weird. You don't want "stars"; you want **specific traits**.

| **Position** | **Archetype / Key Stats**       | **Why?**                                               |
| :----------- | ------------------------------- | ------------------------------------------------------ |
| **QB**       | **Scrambler** (SPD, ACC, CAR)   | Carrying (CAR) is critical. Fumbles kill this.         |
| **HB/SB**    | **All-Purpose** (SPD, ACC, CTH) | Speed kills on the pitch. Needs to catch occasionally. |
| **FB**       | **Power Back** (TRK, STR, CAR)  | The engine. 20 carries/game inside.                    |
| **OL**       | **Agile** (ACC, IBL, RBS)       | Need speed to down-block LBs. Size doesn't matter.     |
| **WR**       | **Blocking** (RBK, IBL, STR)    | If they can't block, they don't play.                  |

If you run the Option, prioritize these archetypes:

| **Position** | **Archetype / Key Stats**          | **Why?**                                           |
| :----------- | ---------------------------------- | -------------------------------------------------- |
| **QB**       | **Scrambler / Athlete** (SPD, ACC) | Must be the best athlete on the field.             |
| **HB**       | **Speed Back** (SPD, ACC)          | For pitches and toss plays.                        |
| **FB**       | **Power Back** (TRK, STR, BTK)     | The engine. Needs to consistently gain 3 yards.    |
| **WR**       | **Deep Threat** (SPD, RBK)         | **BLOCKING IS #1.** Secondary skill: Speed for PA. |
| **OL**       | **Run Blocker** (RBK, RBP, STR)    | They dictate the game.                             |

## Minimal glossary 📌

- **Dive:** the inside handoff part of triple option.
- **Pitch relationship:** spacing between QB and pitch back.
- **Constraint play:** call that punishes a specific defensive adjustment.
- **Scrape exchange:** defensive technique to defeat read option.

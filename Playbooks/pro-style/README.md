# Pro Style 🧱 (CFB26 Playbook Style)

## Quick definition 🧠

“Pro Style” is a **balanced offense** that wants to:

- 🏃 Run with multiple tight-end / fullback looks
- 🚀 Use play-action to create explosive passes
- ♟️ Control game flow with formation diversity

```text
       WR                  OL   TE
                                     WR
              TE           QB
                           RB
```

In video game terms, Pro Style is often the best “do everything” identity if you enjoy **huddling 🗣️, shifting 🔄, and setting up tendencies 📉**.

## The identity (one sentence) 🎯

**Run the ball with physicality 🔨, force the defense into the box 📦, then punish them with play-action and isolation routes 🧨.**

---

## Pros ✅ and cons ⚠️

### Pros

- 🕰️ **Game-script friendly**: works while ahead, tied, or behind.
- 🟥 **Great red zone offense**: condensed field favors power, play-action, and TE routes.
- 🦄 **Personnel mismatch machine**: 12/13/21 personnel forces defenses to declare.
- 💣 **Play-action explosives**: deep crossers, posts, and seams are natural complements.
- ⏱️ **Clock control**: you can shorten games when needed.

### Cons

- 🧘 **Requires patience**: you’re “setting things up” with series-based calling.
- 🐢 **Can feel slow**: if your OL struggles, you get stuck in 2nd-and-8 forever.
- 🛡️ **Needs protection**: long-developing play-action is risky vs elite pass rush.
- 🥊 **Needs isolation winners**: must have a WR who wins 1-on-1 when they load the box.

---

## Typical personnel (who’s on the field) 🧩

Pro Style playbooks commonly major in heavier groupings:

- **12 personnel**: 1 RB, 2 TE, 2 WR
  - The “modern pro” base: multiple run looks + strong play-action.
- **21 personnel**: 2 RB, 1 TE, 2 WR (often includes a FB)
  - Classic I-Form / Strong-I feel, great for downhill runs.
- **11 personnel**: 1 RB, 1 TE, 3 WR
  - Used for 2-minute, obvious pass, or to lighten the box after pounding.
- **22 personnel**: 2 RB, 2 TE, 1 WR
  - Short yardage and goal line.

### Personnel cheat table

| Personnel | On-field           | What it’s best for in Pro Style                         |
| --------- | ------------------ | ------------------------------------------------------- |
| 12        | 1 RB / 2 TE / 2 WR | Modern base; credible run looks + strong play-action    |
| 21        | 2 RB / 1 TE / 2 WR | Downhill runs; lead blocks; short yardage and goal line |
| 11        | 1 RB / 1 TE / 3 WR | Comeback/2-minute; spreads out stacked boxes            |
| 22        | 2 RB / 2 TE / 1 WR | True heavy package; goal line / “get 2 yards” football  |

### Role archetypes

- **QB:** strong play-action, good mid/deep accuracy, can throw on the move.
- **RB:** vision + contact balance; must fall forward.
- **FB/H-back (if present):** lead blocker + checkdown.
- **TE1:** in-line blocker who can still catch.
- **TE2:** move TE / H-back who can run crossers.
- **WR1:** isolated route winner (dig/comeback/fade).
- **WR2:** dependable chain mover.

## Typical formations 📐

“Pro Style” in EA-style games usually includes a mix of:

- **I-Form** (normal, tight, twins)
- **Singleback** (ace, wing, bunch)
- **Strong / Weak** (two-back sets)
- **Pistol** (in some pro-style books)
- **Gun** (11 personnel packages)

### Formation families (quick pick table)

| Family                 | Why it’s in Pro Style                                     | When to lean on it                        |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |
| I-Form / Strong-I      | Downhill identity; easy short-yardage calls               | Ahead / short yardage / red zone          |
| Singleback (Ace/Wing)  | Versatile; lots of run looks + boot/play-action           | Neutral script / early downs              |
| Strong/Weak (two-back) | Forces box fits and personnel responses                   | When you want mismatch via FB/H-back      |
| Pistol (if available)  | Keeps downhill run feel with quicker shotgun access       | Mixed tempo; QB run tags if you have them |
| Gun 11 (packages)      | Spreads out stacked boxes; clean 3rd down / 2-minute menu | When behind / long down-and-distance      |

### A simple structure that works

- Major: Singleback + I-Form
- Minor: Gun 11 (comeback when down two scores)

## Core run game families 🏃

You’ll be most consistent if you pick **two run families** and build everything off them.

### Run family cheat table

| Family                 | Why you want it                                           | Best complements                 |
| ---------------------- | --------------------------------------------------------- | -------------------------------- |
| Inside zone / split    | Stay on schedule; works from almost any personnel         | Play-action crossers; boot flood |
| Power / counter        | Defined angles; physical identity; great in short yardage | Boot/play-action; shot concepts  |
| Stretch / outside zone | Forces pursuit; creates cutbacks                          | Play-action keepers; TE leak     |
| Duo (if available)     | Downhill double teams; “zone look, power feel”            | PA seams/corners; quick flats    |

### 1) Inside zone / split zone

**What it gives you:** a consistent “on schedule” run that works from many sets.

```text
      DE  DT     DT  DE
      O   O   C   O   O   TE
              QB        (Split Flow)
              RB  ↘️
```

**In-game coaching points:**

- Read the front; don’t sprint into the A-gap every time.
- If the defense pinches, bounce to the cutback.

### 2) Power / Counter

**What it gives you:** gap-scheme displacement and predictable blocking angles.

**Why it matters in Pro Style:** gap schemes pair extremely well with **play-action boots**.

### 3) Stretch / outside zone

**What it gives you:** forces pursuit and creates cutback lanes.

**Best used:** when you have speed at RB and a TE who can seal.

### 4) Duo (if in your playbook)

“Duo” is a downhill, double-team run that plays like inside zone but hits like power.

## Core pass game families 🧨

### Pass family cheat table

| Family / concept         | What it gives you                | When to call                                    |
| ------------------------ | -------------------------------- | ----------------------------------------------- |
| PA deep cross / Yankee   | Explosives off LB/safety trigger | After successful runs; 1st down near midfield   |
| Boot / rollout flood     | Easy layered read; move pocket   | Early downs; vs heavy rush; “need a completion” |
| Dagger (dig + clear-out) | Reliable intermediate chunk      | 3rd & 6–10; vs two-high                         |
| Smash / corners          | Sideline + red zone beater       | Red zone; vs Cover 2 looks                      |
| Screens/checkdowns       | Blitz punish + drive sustain     | vs pressure; when defense overplays deep shots  |

### 1) Play-action deep cross / Yankee concepts

**What it is:** a deep crosser paired with a post over the top.

**Why it works:** linebackers step up; safeties get stressed horizontally and vertically.

**When to call:**

- after 2–3 successful runs
- 1st down near midfield
- 2nd & short

### 2) Boot / rollout flood

**What it is:** moving pocket with a layered route (flat + intermediate + deep).

**Why it works:** cuts the field in half; gives easy reads.

**Key coaching point:** don’t drift into sacks—set the edge and throw on time.

### 3) Dagger (dig + clear-out)

**What it is:** a deep dig behind a vertical stem.

**When to call:**

- vs two-high when you need an intermediate chunk
- on 3rd & 6–10

### 4) Smash / corner routes

**What it is:** corner + hitch (or corner + flat).

**Why it works:** a classic red zone / sideline beater.

### 5) Screens and checkdowns

Pro Style doesn’t have to be “deep or nothing.” Use:

- RB swing / flat routes
- TE delay / leak
- WR quick screens when corners play soft

## Building your Pro Style call sheet (practical CFB26 plan) 🗂️

### Situational call sheet table

| Situation   | Go-to calls (examples)                                               | Point of emphasis                       |
| ----------- | -------------------------------------------------------------------- | --------------------------------------- |
| Early downs | Inside zone, power/counter, boot flood, PA cross (sprinkle)          | Stay on schedule; force safety trigger  |
| 2nd & short | PA shot (post/cross), max protect dig, outside zone PA               | Attack; don’t “waste” advantage         |
| 3rd down    | 1–2: power/sneak/flat • 3–6: boot/smash • 7+: dagger/deep cross/mesh | Know your money concept by distance     |
| Red zone    | Heavy run, PA TE corner/leak, fades/stops, bunch rubs                | Condensed field: win leverage quickly   |
| Backed up   | Inside zone, boot to flat, safe quick game                           | Avoid sacks/TFLs; take easy completions |

### Early downs (establish identity)

- Inside zone (12 or 21)
- Power/counter
- Stretch (sprinkle)
- Play-action cross (shot)
- Boot flood (easy completion)

**Rule of thumb:** if you can average 4–5 yards on early-down runs, the passing game becomes easy.

### 2nd and short (attack)

- Play-action shot (post/cross)
- Max protect deep dig
- Outside zone play-action (if available)

### 3rd down

- 3rd & 1–2: power, sneak, quick flat to FB
- 3rd & 3–6: boot flood, smash, TE option
- 3rd & 7+: dagger, deep cross with checkdown, mesh (from gun package)

### Red zone

- Heavy run (force goal-line looks)
- PA TE corner / TE leak
- Fade/stop routes to your WR1
- Bunch/stack rubs if your book includes them

### Backed up

- Inside zone
- Boot with immediate flat
- Safe quick game from gun (if needed)

## “If they do X, I do Y” (defensive answers) 🛡️

| Defensive look/problem      | Your answer                                 | Notes                                    |
| --------------------------- | ------------------------------------------- | ---------------------------------------- |
| Light box (two-high, dime)  | Run until they stop it; add stretch         | Make them add a fitter                   |
| Aggressive LBs crashing run | PA crossers/seams; leak opposite run action | Throw where they vacated                 |
| Cover 3 / single-high zone  | Flood/deep outs; posts if you can protect   | Stress the seams and outside thirds      |
| Heavy blitz tendency        | Max protect + 1-on-1 shot; screens          | Don’t drift; take the cheap answer early |

### They play light box (two-high, dime)

- Run until they stop it.
- Mix in stretch to punish wide alignments.

### They crash run with aggressive LBs

- Play-action crossers and TE seams.
- Leak the TE/RB opposite the run action.

### They sit in Cover 3 / single-high

- Flood concepts and deep outs.
- Post routes behind the middle safety if you can protect.

### They blitz a lot

- Use max protect.
- Keep the RB/TE in and take your one-on-one shot.
- Call screens to slow them down.

## Sequencing (how Pro Style creates explosives)

Pro Style is at its best when you intentionally build tendencies and then break them.

### The simplest tendency/counter pair

- Tendency: inside zone / power from 12 or 21
- Counter: play-action deep cross / post

If you call the counter before the tendency has forced the defense to react, it feels like “PA doesn’t work.” If you call it after you’ve forced a safety to trigger, it becomes a huge play generator.

### A practical 8-play “setup” series

1. Inside zone (12)
2. Power (21)
3. Boot flood (off the last run look)
4. Inside zone again (12)
5. Stretch (punish overloading inside)
6. Play-action cross (shot)
7. Checkdown concept (keep drive alive)
8. Run to finish the series

This is not a rigid script—think of it as a pattern you can repeat during a game.

## Personnel packages (what to major in and why)

Pro Style succeeds when you can force the defense to pick a poison:

- stay in base and risk getting beat by play-action/isolations
- go lighter and risk being run over

### Package 1: 12 personnel “modern pro” base

Best for:

- inside zone + split zone
- play-action crossers
- TE seams and corners

### Package 2: 21 personnel “downhill” package

Best for:

- power/counter
- play-action boot
- short yardage and goal line

### Package 3: 11 personnel comeback package

Best for:

- playing from behind
- 2-minute
- spreading a defense that is selling out to stop your heavy packages

## Audible package (so you can punish the look)

Pro Style audibles should cover:

- heavy box (throw quickly)
- light box (run)
- blitz (screen or quick throw)
- two-high (run or intermediate concept)

Suggested audible set (example):

1. Power / inside zone (base)
2. Quick slant/flat (answer vs heavy box)
3. Boot flood (easy completion)
4. Play-action shot (post/cross)
5. Screen (blitz punish)

Pre-snap checklist:

- If they’re in dime: run it.
- If corners are off: take quick game.
- If safeties are creeping: shot play.

## Clock and field-position plan (a Pro Style advantage)

If you like playing “real football,” Pro Style is excellent at controlling game state.

- **With a lead:** run on early downs, throw off play-action on 2nd & short, and avoid turnovers.
- **In a tie game late:** target efficient completions to TEs/RBs and stay in bounds.
- **When backed up:** call your safest run and your fastest throw; avoid slow-developing PA.

## Troubleshooting

### Problem: “I can’t run the ball.”

Fix:

- run toward your best OL/TE surface
- use motion to get better angles
- mix in stretch to punish pinching

### Problem: “Play-action gets me sacked.”

Fix:

- don’t call long PA on obvious pass downs
- keep extra protectors in (TE/RB)
- take the checkdown and live for 2nd & 6

### Problem: “My WRs never get open.”

Fix:

- throw more to TE/RB (the pro game is built on this)
- use bunch/stack to defeat press
- call isolations only when you like your matchup

## Run-game diagnostics (how to know where to run)

Pro Style becomes much easier when you stop calling runs “randomly” and start aiming them at defensive structure.

### A simple run-direction rule

- If the defense is **over-shifted** to one side (extra body outside your TE), run back the other way.
- If the defense is **light inside** (fewer bodies between the tackles), run inside zone/duo.
- If the defense is **pinching** hard, call stretch or counter.

### What to do when they are run-blitzing

Signs:

- linebackers shooting gaps immediately
- safeties filling fast on early downs

Answers:

- use play-action leak
- run counter (let them run themselves out of the play)
- call quick slant/flat on 1st down (cheap yards)

### Don’t forget “run to set up the run”

If you run inside zone 6 times in a row, the defense will adjust. Mix in:

- power
- counter
- stretch

Even if the yardage is similar, the defender fits change, and that’s what creates mistakes.

## Short-yardage mini-playbook

If you want Pro Style to feel dominant, decide your “3 calls” for 3rd & 1–2.

Suggested menu:

- Power (behind your best surface)
- Inside zone/duo (if the defense spreads out)
- Boot flat (if the defense sells out)

In games, short yardage is as much about avoiding negative plays as it is about getting 20-yard touchdowns.

---

## Pace note (Pro Style can still go fast)

You don’t have to huddle every snap. A simple approach:

- Go no-huddle after a successful run to keep the same defensive look.
- Slow down after penalties/TFLs to reset.

This keeps Pro Style from feeling “slow,” while still letting you control the game when you have a lead.

## Minimal glossary

- **Gap scheme:** linemen block down while a puller kicks/wraps (power/counter).
- **Zone scheme:** linemen step together; RB reads leverage and cuts.
- **Flood:** three routes at different depths to one side.
- **Yankee:** deep post + deep crosser.

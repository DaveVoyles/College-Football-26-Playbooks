## Quick start 🚀

1. Open the index: [Playbooks/README.md](Playbooks/README.md)
2. Pick a scheme family.
3. Favorite a small set of plays using the guide’s “situational favorites” section.
4. Add teams you confirm in-game to the scheme’s **CFB26 team list**.

## What’s inside each playbook folder 📁

Each scheme guide includes:

- Pros/cons ✅⚠️
- Base personnel + formation families
- Core concept menu + audibles
- Situation plan (early downs, 3rd down, red zone, 2-minute)
- Troubleshooting (common failure modes)
- **Teams to check in CFB26** (real-world programs commonly associated with the style)
- **NFL analogs** (teams that commonly use similar principles)

## About “teams that use these playbooks” (important) 🔎

CFB26 team-to-playbook assignments can change with patches/roster updates.

So each scheme guide includes two things:

1. A **“Teams to check first”** list (based on real-world scheme association)
2. A **fillable table** for “Teams in CFB26 that use this playbook” once you verify it in-game

Recommended workflow:

1. In CFB26, open team selection / playbook settings.
2. Note the offensive playbook label for a team.
3. Add it to the matching scheme’s “Teams in CFB26” table.

If you tell me your mode (**Dynasty / Road to Glory / Play Now**) and whether you want **realistic** vs **meta**, I can tune the gameplans accordingly.
